# Текущая Запись

## Статус

Этот файл относился к старому сценарию автоматической брони.
Теперь `/search` работает только как постоянный мониторинг свободных слотов.
Бот не удерживает слот и не ждёт СМС-код.

## Что Делать

Запустите мониторинг в Telegram:

```text
/search 17:00-22:00
```

Когда бот пришлёт свободный слот, записывайтесь вручную на сайте PADL.
Чтобы остановить мониторинг:

```text
/stop
```
