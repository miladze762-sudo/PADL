# Запуск Бота Для Мониторинга

## Текущий Режим

`/search` теперь работает как постоянный мониторинг свободных слотов.
Бот не удерживает слот, не запрашивает СМС и не подтверждает бронь.
После уведомления нужно записываться вручную на сайте PADL.

Текущий процесс бота:

```text
PID: 6200
Статус: мониторинг, новых слотов нет
```

## Что Сделать Сейчас В Telegram

1. Откройте Telegram.
2. Найдите бота `@PADL77bot`.
3. Нажмите `Start` или отправьте:

```text
/start
```

4. Затем отправьте:

```text
/status
/search
```

Бот будет присылать только новые найденные слоты и продолжит мониторинг до команды `/stop`.

## Почему Бот Может Не Отвечать Первым

Если Telegram возвращает `Bad Request: chat not found`, обычно пользователь ещё не
нажал `Start` у нового бота. После вашего `/start` бот сможет отвечать.
