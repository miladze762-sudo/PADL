# Статус Подготовки

## Что Уже Сделано

- Создан файл `.env` в папке проекта.
- Telegram bot token и `ADMIN_CHAT_ID` уже записаны.
- Виртуальное окружение `.venv` создано.
- Проект переведён на режим постоянного мониторинга: `/search` только уведомляет о свободных слотах.
- Профиль для старой автоматической брони может оставаться в `data/padlbot.db`, но мониторинг его не требует.

## Что Делать В Telegram

1. Откройте Telegram-бота `@PADL77bot` и нажмите `Start`.
2. Отправьте:

```text
/status
/search
```

3. Когда бот пришлёт свободный слот, записывайтесь вручную на сайте PADL.
4. Чтобы остановить мониторинг, отправьте `/stop`.

## Как Запустить Бота

```powershell
.\.venv\Scripts\Activate.ps1
python -m padlbot
```

## Как Проверить Доступность Из Терминала

```powershell
.\.venv\Scripts\python.exe scripts\dry_run.py
```

## Android SMS Forwarder

Для нового режима мониторинга Android SMS-forwarder не нужен.
