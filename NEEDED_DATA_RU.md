# Какие Данные Нужны От Вас

Для нового режима постоянного мониторинга нужен только Telegram-бот.
Данные для брони, Android SMS-forwarder и `PADL_DRY_RUN` больше не нужны для `/search`.

## 1. Telegram Bot Token

Что указать в `.env`:

```text
TELEGRAM_BOT_TOKEN=...
```

Как получить:

1. Откройте Telegram.
2. Найдите бота `@BotFather`.
3. Отправьте `/newbot`.
4. Придумайте имя бота, например `PADL Monitor Bot`.
5. Придумайте username, который заканчивается на `bot`.
6. BotFather пришлёт токен вида `123456789:AA...`.
7. Скопируйте этот токен в `.env`.

## 2. Ваш Telegram Chat ID

`ADMIN_CHAT_ID` нужен только для автозапуска мониторинга через `AUTO_START_SEARCH=1`.
Если вы запускаете мониторинг вручную командой `/search`, его можно не заполнять.

Как получить:

1. Откройте Telegram.
2. Найдите `@userinfobot` или `@RawDataBot`.
3. Нажмите Start.
4. Бот покажет ваш `id`.
5. Скопируйте число в `.env` при необходимости.

## Команды

```text
/start
/search 17:00-22:00
/status
/stop
```

После уведомления о свободном слоте записывайтесь вручную на сайте PADL.
